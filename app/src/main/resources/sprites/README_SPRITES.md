# MineMaze Sprite Requirements

The following sprite images are needed for the MineMaze game. Current placeholders should be replaced with proper 30x30 pixel PNG images:

## Required New Sprites:
1. **wall.png** - Gray brick wall texture (impassable barrier)
2. **bomb_marker.png** - Red crosshairs or target symbol on transparent background
3. **booster.png** - Blue/green power-up symbol (lightning bolt or star)
4. **heavy_rock.png** - Dark gray rock with metallic highlights
5. **bomber.png** - Orange/red vehicle with bomb symbol

## Existing Sprites (to keep):
- pusher.png - Player-controlled pushing machine
- ore_0.png - Ore block (normal state)
- ore_1.png - Ore block (on target)
- rock.png - Regular destructible rock
- target.gif - Target zone marker

## Removed Sprites (no longer used):
- bulldozer.png - Removed from MineMaze
- clay.png - Removed from MineMaze
- excavator.png - Removed from MineMaze

## Design Guidelines:
- All sprites should be 30x30 pixels
- Use PNG format with transparency where appropriate
- Keep consistent art style with existing sprites
- Use clear, distinguishable colors and shapes
- Ensure good visibility on the game grid background
